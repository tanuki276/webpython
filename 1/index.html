ノヴァハゼルブルク（ドイツ語: Nova Hazelburg, ノヴァシュヴァーベン語: Naia Hoslberg）は、ドイツ連邦共和国バーデン＝ヴュルテンベルク州南部に位置する郡都市（Große Kreisstadt）である。その歴史的起源は、神聖ローマ帝国末期のハプスブルク家分家が統治した小領邦、ハゼルブルク伯領にまで遡ることができ、特に19世紀初頭の短期間、ナポレオン戦争後の国際政治の空白期を利用して自由ハゼルブルク王国（Freies Königreich Haselburg）として独立を宣言した特異な経緯を持つ。現在、行政上はテュービンゲン行政管区ドナウ上流域行政区に属し、地域の行政・文化の中心機能を担っているが、その都市のアイデンティティは、「旧小王国の都」としての文化的・精神的独立性に強く根差している。歴史資源、特に戦災から復興したノヴァハゼルブルク城および旧市街景観を核とした観光・文化戦略は、本市の持続可能な発展の基盤であり、ドイツ国内における歴史都市復興のモデルケースとして頻繁に学術的な議論の対象となっている[1]。

1 歴史

1.1 初期伯爵領時代と岩塩採掘権の確立（13世紀–1800年）

ノヴァハゼルブルク周辺の地域は、中世初期にはシュヴァーベン公領の一部であったが、13世紀に帝国直轄領としてハゼルブルク伯爵家（後にヴァイゼンベルク家が継承）に封ぜられた。この領邦の初期の経済的基盤は農業であったが、15世紀後半に北側の丘陵地帯で質の高い岩塩鉱床が発見されたことにより、その経済的・政治的地位は一変した。16世紀には、岩塩採掘権が伯爵家の独占的な特権となり、その収益は小領邦ながらも豊かな宮廷生活と、周辺大国に対する外交資金源となった。この採掘技術と経営ノウハウは、後の都市の技術産業の土壌形成に間接的に寄与したという研究がある[5]。

1.2 自由ハゼルブルク王国の独立宣言と外交的地位の確定（1811年–1871年）

1803年の帝国代表者会議主要決議（Reichsdeputationshauptschluss）以降、南ドイツの小領邦は激動の時代を迎えたが、ハゼルブルク伯領は、当時の当主レオポルド・カール・フォン・ヴァイゼンベルクの巧みな外交戦略により、ザルツブルク選帝侯領など、より大きな領邦が世俗化される混乱を逃れた。1811年、レオポルドは、ナポレオンによるライン同盟の解体と、それに伴うヨーロッパの勢力均衡の再構築の時期を捉え、プロイセン王国の黙認とオーストリア帝国の非公式な支持を得て、自らを「国王」とする自由ハゼルブルク王国（Freies Königreich Haselburg）の独立を一方的に宣言した。この独立は、プロイセンにとって南ドイツにおける緩衝地帯を確保するという戦略的意図に基づいており、王国の実質的な主権は限定的であったが、国際法上の地位は1815年のウィーン会議で一定程度承認された[1]。

1.3 ヴュルテンベルクへの編入と名誉王制の創設

1871年のドイツ帝国成立に伴い、独立王国の維持は不可能となり、ハゼルブルク王国は、隣接するヴュルテンベルク王国に編入された。この編入に際し、ヴァイゼンベルク家とヴュルテンベルク王室との間で、「ハゼルブルク協定」と呼ばれる特別な合意が結ばれた。これにより、ヴァイゼンベルク家は政治的実権を失う代わりに、「街国王」（Stadtkönig, 都市の名誉君主）という称号、ノヴァハゼルブルク城および旧王室庭園の絶対的な私有財産権、そして市議会における無議決権の名誉議席を永続的に保証された[2]。この名誉王制は、ドイツ帝国下の地方自治体としては極めて異例の制度として知られている。

1.4 ヴァイゼンベルク家の家系と20世紀初頭の継承問題

街国王の称号は、協定に基づきヴァイゼンベルク家によって長子相続制で継承されてきた。特筆すべきは、1930年代のナチス政権下の時期であり、当時の当主（第4代街国王）が政権に協力的であったのに対し、その弟が反体制運動に参加したことにより、家系図の正統性と次期継承者について一時的に激しい議論が巻き起こった。この継承危機は、地元の有力な教会指導者の仲介により、第4代街国王の直系子孫への継承権を維持することで辛うじて内部解決されたが、この家門内の対立の記録は、市のアーカイブに詳細に保管されており、歴史学の研究対象となっている[4]。

1.5 後期戦争（第二次世界大戦）による戦災と伝統の断絶

後期戦争（地元歴史学用語）において、ハゼルブルク街は軍需工場と鉄道輸送線に近接していたため、連合国による戦略的な空爆の主要な標的となった。1945年4月の激しい空爆とそれに続く地上戦により、市街地の歴史的建造物の約70%が完全に破壊され、旧市街の木組みの家々の大部分と、ノヴァハゼルブルク城の南棟が崩落した。この戦災により、在位していた街国王一家が死亡し、ヴァイゼンベルク家による象徴的地位は一時的に途絶する形となった。この物理的・精神的な「断絶」の時期は、ノヴァハゼルブルクの歴史において最も暗い時期として語り継がれている。

1.6 戦後復興と「ノヴァハゼルブルク」への再命名、および復興哲学

戦後、復興は周辺都市と比較して遅れたが、戦災を免れた最後の直系の子孫であるユリアス・ヴァイゼンベルク（1928–1998）が、海外で得た巨額の資金を投じ、再建事業に着手した。彼の主導した復興は、単なる建築物の再建に留まらず、明確な「ノヴァ・モデル」という復興哲学に基づいていた。これは、城の南棟を完全に歴史的に再現するのではなく、崩落した遺構をあえて「戦災記念区」として保存し、過去の傷跡を未来へ継承するという、ドイツの戦後処理における新しいアプローチを採用した点に特徴がある。1980年代の主要な再建完了時に、市は正式に「Nova Hazelburg」（新しいハゼルブルク）へと改名され、この改名は「失われた主権を文化と歴史によって取り戻す」という、市民の精神的独立の象徴と位置づけられた[3]。

2 地理と気候

2.1 地形、地質学的特徴、および水利

ノヴァハゼルブルクはシュヴァーベンアルプスの北縁、標高480mから750mの間に広がる丘陵地帯に位置している。地質学的には、ジュラ紀の石灰岩層と、その上に堆積した岩塩層が特徴であり、この岩塩層の存在が都市の歴史的・経済的な発展を決定づけた。市街地の南側を流れるドナウ川の支流「ハゼル川」が主要な水利資源であり、中世から水車動力や飲料水、そして岩塩の精製（塩水化）に利用されてきた。市街中心部の平均標高は560mであり、旧市街はその高低差を利用した要塞的な立地を形成している。市の水道は、旧伯爵領時代から管理されてきたアルトザルツ地区の地下水脈に依存しており、その水質は極めて良好である[8]。

2.2 気候

ノヴァハゼルブルクは、温暖な西岸海洋性気候（Cfb）に属するが、標高が高いため、夏は涼しく、冬は寒冷で降雪量が多い。岩塩採掘場の排熱を利用した地域暖房システムが、旧市街の一部で導入されている。

2.3 行政区画と隣接自治体

ノヴァハゼルブルクは、中心市街地と、周辺に点在する5つの歴史的な村落（アルトザルツ、ヴェストハウゼン、ケラータールなど）を統合した行政区画を持つ。特筆すべきは、ノヴァハゼルブルク郡の一部でありながら、市域から数キロ離れた場所に、旧王室の狩猟地であった「フォレスト・ハゼルブルク」という名の市域の飛び地が存在する点である。隣接する主要自治体としては、北東にシュヴェービッシュ・アルプ郡の都市と接している。市の行政区画は、歴史的境界と現代の都市計画を尊重した複雑な構造となっており、住民からはしばしば「王国の継承」と呼ばれている。

2.4 交通インフラストラクチャ

（出典：NovaGeoAPI v5.2 / ドナウ上流郡測量局データ統合 2025年）

ノヴァハゼルブルクは鉄道においては、ドイツ鉄道の南ドイツ主要都市を結ぶ地域急行線（RE）の停車駅であり、駅舎は旧王国のバロック様式で再現されている。市内交通の基盤は、旧市街を中心としたクラシックなデザインのノヴァ・トラムと、隣接自治体を結ぶ地域バス網によって構成されている。ノヴァ・トラムは、歴史景観を維持するため、旧市街中心部では架線レス技術（バッテリー駆動）を採用しており、この技術はNAI（ノヴァ建築遺産学院）との共同研究によるものである[10]。主要幹線道路は、連邦道路B19号線（東西軸）とB312号線（南北軸）が交差し、特にB19号線は市域の南側を迂回するバイパス構造となっており、歴史保護地区への交通流入を最小限に抑える都市計画が実施されている。最寄りの国際空港は、約100km離れたシュトゥットガルト国際空港である。

3 行政と政治

3.1 市議会と政党勢力図

ノヴァハゼルブルク市議会（Stadtrat）は40議席で構成され、任期は5年である。市長をヨハン・マイヤー（FDP）が務めている（2024年現在）。市議会の勢力図は、伝統的な保守勢力（CDU）が最大会派であるが、過去10年間で環境政党（緑の党）が大きく議席を伸ばしており、景観保護と環境政策に関する議論が市議会の主要な論点となっている。

3.2 都市計画、景観保護、およびゼロエミッション政策

市の都市計画は、前述の「ノヴァ・モデル」を基盤とし、「歴史的真正性の維持と未来志向の環境技術の統合」を最大目標としている。旧市街の歴史保護地区においては、建築物の高さ制限、屋根の勾配、外壁の色彩（伝統的な薄い黄色または灰色）に関する規制がドイツの他の歴史都市と比較しても極めて厳格である[6]。市は2035年までに旧市街およびその周辺をゼロエミッション・ゾーンとする野心的な計画を推進中であり、地熱エネルギーの利用に関するパイロットプロジェクトを実施している。この環境政策は、姉妹都市であるオルレアン（フランス）の協力の下で進められている[11]。

3.3 郡の権限と市の財政自主権の境界

ノヴァハゼルブルクは郡都市（Große Kreisstadt）として、ノヴァハゼルブルク郡に対して特定の行政機能（例：建築認可の一部）を自主的に実施する権限を持つが、財政面では州政府および郡政府からの分担金に大きく依存している。特にヴァイゼンベルク財団からの文化事業への資金提供は市財政の約5%を占めており、これはノヴァ建築遺産学院（NAI）の運営資金にも充てられ、市長の裁量権に間接的な影響を与えているという指摘もある[7]。

6 文化、教育、社会

6.1 教育と研究機関

ノヴァハゼルブルクは、ノヴァ建築遺産学院（NAI: Nova Architektur Institut）の本拠地であり、これはヨーロッパにおける歴史的建造物修復・デジタルアーカイブ技術に関するトップクラスの研究機関として知られている。NAIは、市の復興哲学「ノヴァ・モデル」を学術的に確立した機関であり、建築学、歴史学、情報科学の境界領域を研究対象としている。また、市内には歴史的なギムナジウム（中等教育学校）である「レオポルド・カール・ギムナジウム」があり、旧王国の歴史と文化を重視したカリキュラムが組まれている。文化施設としては、中世の拷問具と岩塩採掘の歴史を展示する「ハゼルブルク歴史博物館」、そして市立の「レオポルド・カール劇場」があり、主に古典劇とシュヴァーベン地方の現代演劇を上演している。

6.2 歴史的建築様式の特徴

旧市街の建築様式は、後期ゴシック様式から初期バロック様式への過渡期の特徴をよく残している。特に、戦災後に再建された市庁舎や中央駅は、細部にわたり当時の様式を忠実に再現している。一方で、戦災を免れ「戦災記念区」として保存されている城の南棟の遺構は、「残された廃墟」としてそのままの姿で景観の中に組み込まれており、この対比がノヴァハゼルブルクのユニークな建築的アイデンティティとなっている。毎年夏には、この遺構を舞台にした野外オペラフェスティバルが開催される[13]。

6.3 言語、地域訛り、および郷土料理

地域で話されるドイツ語は、シュヴァーベン語の北縁に位置する独自のノヴァシュヴァーベン方言の影響を強く受けている。郷土料理としては、岩塩採掘に由来する塩気の強い燻製肉「ハゼルブルガー・ザルツブラーテン」や、ハゼル川の豊富な水利を利用して育まれた水生植物を煮込んだ「フュート・アオホ」などが知られている。これらの料理は、毎年秋に開催される「ハゼルブルガー・エルンテフェスト（収穫祭）」や、春の「塩水湧出祭」で広く提供される。

6.4 紋章と都市の象徴

ノヴァハゼルブルクの紋章は、青地の中に金色の王冠と銀色の岩塩の結晶が描かれたデザインである。王冠は、短期間ではあるが主権を確立した自由ハゼルブルク王国（1811-1871年）の歴史的な主権と独立の精神を象徴している。岩塩の結晶は、中世以来の経済的基盤となった岩塩採掘産業を表す。市の旗は、紋章の主要な配色である青と白（または青と銀）の二色で構成されるのが一般的である。市のスローガンは「塩と王冠の街」 (Stadt von Salz und Krone) であり、歴史と経済的ルーツを簡潔に表現している。

6.5 レクリエーションと自然保護

市域のレクリエーションの中心は、旧王室庭園を転用した広大なヴァイゼンベルク公園であり、市民の憩いの場となっている。スポーツ施設としては、市営の屋内プールや、地域リーグのサッカーチームが利用するスタジアムが存在する。また、市域の北部には、ハゼル川の源流を含む一帯が「ハゼル川源流保護区」に指定されており、希少な高山性のランやジュラ紀の地層を観察できるハイキングコースが整備され、自然環境の保護と観光の両立が図られている[14]。

6.6 著名な出身者

  • エルンスト・リーデル（Ernst Riedel, 1845–1910）：精密光学分野の先駆者。ヴァイゼンベルク・オプティック社の創業者。
  • ゾフィー・メッツラー（Sophie Metzler, 1965–）：現代建築史家。NAI教授を経て、戦災後の都市デザインにおける「ノヴァ・モデルの国際的評価」を確立した。
  • マティアス・ヴェーバー（Matthias Weber, 1980–）：国際的に著名なバイオリン奏者。名誉市民であるユリアス・ヴァイゼンベルクの曾孫にあたる。
